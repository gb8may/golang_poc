# NetCatcher
My first POC using GoLang

```
# go build
```
Get IP from websites:
```
# ./net-catcher ip --host website.ca
```

Get server name from websites:
```
# ./net-catcher servers --host website.ca
```

Get help:
```
# ./net-catcher -h
```
